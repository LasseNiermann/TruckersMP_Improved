How to get Steam Web API Key?

1) Go to http://steamcommunity.com/dev/apikey

2) Paste some random site, check I agree and click Register

3) Copy Steam Web API Key

4) Paste it right there

5) Click "Save Settings" in navigation menu and reload TruckersMP page.

Template: number, key, title (optional);. Use ; to delimite buttons, , to delimite options.

Example 1: 3,d; will generate "+3 days" button with +3 days trigger.
Example 2: 7,d,One week; will generate "One week" button with +7 days trigger.
